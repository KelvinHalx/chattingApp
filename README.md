# Chatting Web App

Entire documentation can be found ![**here**](https://github.com/saharshleo/chattingApp/blob/master/docs/Chat%20App%20Doc.pdf)

## Current Features:
* User Authentication
* Global chat room
* Public chat rooms on particular topics
* Peer to peer chat
* Storing chat history of peer to peer chats

## Tools Used:
1. HTML, CSS
2. Bootstrap CSS
3. JavaScript
    * Socket-IO Library
4. Python libraries and frameworks
    * Flask web framework
    * Flask SQLAlchemy
    * Flask Bcyrpt
    * Flask Socket IO
    * Flask WTForms
    * Flash Login
